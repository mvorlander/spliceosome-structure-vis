# Spliceosome Cryo-EM ChimeraX Scripts

Lightweight static dashboard for copying ChimeraX scripts for spliceosome cryo-EM PDB entries.

- 131 deposited PDB entries.
- Original deposited chain identifiers only.
- No maps, mmCIF files, local CIF models, or large binary assets are included.
- ChimeraX scripts use `open <pdb_id>` so ChimeraX downloads structures directly from the PDB.

Open `index.html` locally, or enable GitHub Pages for this repository to serve the dashboard from the repository root. GitHub Pages is not available for this private repository on the current account plan; after making the repository public, enable Pages with source `main` and folder `/`.

The local curated CIF-only entries `9s2e` / ILS-AQR, `local_step2`, and `9s2f` / DIS are intentionally excluded.
